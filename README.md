# MD1-1
## 動作
・DCモーターを一つ、動かせます

・耐圧に関しては搭載する部品にもよるので、なんとも言えません。

・ニッケル水素電池12Vにも対応できるように工夫をしたつもりです。

## 仕様
・フォトカプラには「TLP152」を使用。

・ゲートドライバには「IR2302」を使用。

・MOSFETのフットプリントを「TO263」にしたので、だいたいのMOFETを積める

└どれも部室にある部品なので、作りやすい

・入力駆動電圧の選択するはんだジャンパ付き

└入部したての一年生教育には
